Class: ApolloClientLink

apollo-stitcher.ApolloClientLink(options)

Class representing a link with management of queries by an Apollo Client layer, for use with server-to-server graphql requests. In particular, when using makeRemoteExecutableSchema, graphql requests are executed directly from a link without an intervening Apollo Client; use of ApolloClientLink opens up the ability to use the Apollo Client cache to reduce unnecessary requests.
